[在线文档](https://www.wueasy.com/docs/wueasy-admin)

## 主要使用技术

* Spring Boot
* Motan
* Dubbo
* ZooKeeper
* MyBatis

##  环境要求

* `JDK 1.8`
* `MAVEN 3.x`
* `ZooKeeper 3.4.13`
* `Mysql 5.7+`、`PostgreSQL 10.0+` 、`MariaDB 10.0+`

## 特性

* 用户登录、退出
* 精细化权限控制，控制到按钮的显示
* 数据权限控制
* 菜单管理
* 用户管理（锁定、注销、激活、重置密码、新增用户）
* 用户安全控制（密码安全加密、用户登录异常锁定、修改密码等）
* 权限管理中心（角色管理、权限分配、数据权限）
* 系统参数管理
* 数据字典管理
* 登录日志
* 操作日志
* 通用导出excel功能
* 拦截器控制（权限拦截器、token拦截器、操作密码验证拦截器）
* 定时任务管理
